# Guuduu

一个简单的响应式静态网页项目，展示一张居中的自适应图片。

## 项目结构

```
guuduu/
├── index.html      # 主页面
├── styles.css      # 样式文件
└── images/         # 图片目录
    └── main-image.jpg  # 主要展示图片
```

## 部署说明

1. 将你想要展示的图片放在 `images` 目录下，并命名为 `main-image.jpg`
2. 项目可以直接部署到 Netlify：
   - 连接你的 GitHub 仓库
   - 选择 main 分支
   - 部署设置保持默认即可

## 特性

- 响应式设计，适配各种屏幕尺寸
- 图片自动调整大小，保持最佳显示效果
- 简洁的页面布局
